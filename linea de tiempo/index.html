Evolución Del Software

Primera etapa

1949-1960

El software estaba en desarrollo y en su conocimiento, los programas eran desarrollados por una persona, era el encargado de escribir el código y las instrucciones. Este era muy complejo y tardaba mucho en su desarrollo. Su distribución fue muy limitada.

Assembly

1949

El lenguaje Assembly sirve para representar de manera simbólica con el uso del código binario, utilizan instrucciones fáciles de entender para la comunicación con el hardware de la computadora, lo que permite controlar el sistema

Fortan

1957

Es un lenguaje de alto nivel, se adaptaba al cálculo numérico y a la computación científica. Lo desarrollo IBM en 1957 fue usado para aplicaciones científica y de ingeniería.

Cobol

1959

Fue un lenguaje universal para cualquier computadora, fue desarrollado en 1959 para ser utilizado en los negocios. Podían realizar documentación, gestión de archivos y de datos.

Segunda etapa

1965-1975

El software era perfeccionado para su comercialización. Tuvieron cambios como, el software permitía multiprogramación, la memoria guarda más de un programa por usuario gracias a los sistemas multiusuarios, es decir varios usuarios existen en una terminal en línea utilizando el modo de operación de tiempo compartido,Pero el costo de mantenimiento era elevado.

OS/360

1965

Fue un sistema operativo de procesamiento también, ademas fue diseñado para el soporte multiprogramación.

Ingeneria De Software

1968

Se creó la ingeniería del software, significa el conocimiento científico de diseño y construcción de los programas para computadoras asociada a la documentación necesaria para desarrollar, operar y mantenerlos

UNIX

1969

UNIX es un sistema portable, multitarea y multiusuario fue echo en los laboratorios Bell de AT&T.Es un sistema diseñado para la creación de un software y el diseño modular.

Multics

1969

Sistema operativo de tiempo compartido para funcionar los siete días de la semana, fue utilizado para un único nivel de almacenamiento de acceso de datos

PASCAL

1970

se estableció un servicio de red de internet con la UCLA, se utiliza el lenguaje de programación PASCAL, para la programación imperativo y procedimental, trabaja de manera rápida y eficiente

Lenguaje C

1972

Es un lenguaje de programación de un nivel medio, por lo cual no necesita de muchos recursos, ofrece una ejecución rápida y eficiente de los programas.

CP/M

1974

Fue uno de los primeros sistemas operativos Servía para ejecutar programas y procesar archivos usando el teclado, utilizando el sistema monousuario y monotarea

Tercera Etapa

1975-1985

El desarrollo de software permitió que sea un producto con un hardware de bajo costo. En estos tipos de sistema se empezaron a utilizar algoritmos que al día de hoy se conocen como inteligencia artificial, así de esta manera se empezó a ampliar las redes de áreas locales y globales. Como también conceptos de productividad y calidad, se relaciona el uso de esta tecnología.

MS-DOS

1981

Se utilizaban comandos para utilizarlos, que provenían de una lista de comandos, se ejecutaban a través del teclado. Se distinguen entre: comandos internos y externos. MS-DOS hecho por Microsoft para IBM PC y otras computadoras que utilizaban la CPU Intel 8088 y sus sucesores, y UNIX.

SunOS

1982

Fue un sistema operativo derivado por Unix y BSD desarrollado por Sun Microsystems, para ser utilizado en lugares de trabajo y servidores a principio de los noventa

C++

1983

Es un lenguaje de programación que fue orientado al lenguje c, sirve para desarrollar software estableciendo sus propios datos corrigiéndolos mediante funciones y métodos, además sirve para el desarrollo apps de escritorio, juegos entre otras.

Mac OS

1984

Era barato debido a su procesador, realizaba procesamiento de textos, base de datos y hojas de calculo. Fue utilizado principalmente por escuelas debido a su accesibilidad, un software educativo.

Cuarta Etapa

1985-2000

Con la llegada del internet la demanda del software fácil de usar fue alta, como el perfeccionamiento de los programas de acuerdo a la manera en que se utilizara, para que fuera optimizado. El sistema es decir el software se empieza a especializarse según el entorno que se requería, genero un sistema cliente-servidor

OS/2

1987

Es un sistema operativo de 16 bits para ser utilizado en el microprocesador 286, tenia un modo de protegido para la computadora, para la protección de la memora, gestión y multitarea, etc.

Python

1991

Es un lenguaje de programación de alto nivel utiliza un sistema de multiparadigma, se usa para desarrollar apps, la ciencia de datos, crear inteligencias artificiales.

GNU/Linux

1991

Es un SO libre de tipo Unix, fue un sistema multi-usuario y multi-tarea. Son un conjunto de programas que sirven para interactuar con la pc y ejecutar otros programas

FreeBSD

1993

Tiene un SO multiusuario, también realiza procesos multitarea y multiproceso, su sistema operativo de UNIX de tipo de software libre.

Microsoft Windows NT

1993

Tenia un Sistema operativo con una interfaz gráfica propia, además de eso ofrecía mayor estabilidad, seguridad y multitarea. El núcleo NT sirve para los sistemas operativos de Windows actuales.

Windows 95

1995

De software a sistema operativo hecho por Microsoft, se caracteriza por su sistema híbrido de 16 y 32 bits, su rendimiento optimo, funciones multimedia de calidad y sus interfaces graficas.

Java

1995

Un lenguaje de programación comercializado por Sun Microsystems. Es rápido y confíale para diseñar servicios y aplicaciones, utiliza motores de procesamiento y un sistema multitarea

Palm OS

1996

Fue un sistema operativo móvil, caracterizado por su interfaz móvil, accesibilidad y funcionabilidad, ofrecía a su usuario una experiencia única aunque rápidamente quedo obsoleto

Windows 98

1996

Un sistema hibrido de 16 y 32 bits, pero ahora tenia un software mucho mas rápido al momento de almacenar datos. Una característica destacada era que usaba menos recursos al momento de ejecutar programas

Quinta Etapa

200- Presente

La demanda por el software fue significativa, se requería un software funcional cada año. Ofrece soluciones tecnológicas accesibles y simples, para que puede utilizarse en cualquier entorno. De esta manera se convirtió en una alternativa fácil de desarrollar, almacenar y recuperar cálculos e información relevante.

Visual Basis.NET

2001

Es un lenguaje de programación, cuenta con una interfaz gráfica, memoria automática, etc. Sirve para desarrollar apps de escritorio, web y móviles. Además, tiene un montón de características para desarrollar apps.

Windows XP

2001

Es un SO que permite utilizar distintas aplicaciones. ES un sistema muy fácil de usar, cuenta con una función multitarea, esto permite ejecutar varios programas a la vez.

Mac OS X

2001

Es un sistema operativo basado en UNIX, que fue desarrollado y comercializado por Apple Inc. introdujo una nueva arquitectura moderna, mayor estabilidad y mejorades interfaces gráficas

ReactOS

2004

Tenía un sistema operativo de código abierto, permitía la compatibilidad binaria con las aplicaciones de software.

Android

2005

Es un sistema operativo móvil, se creó utilizando como base Linux y otros software. Diseñado para teléfonos, tablets y relojes inteligentes

SymbOS

2005

Este sistema operativo fue diseñado para amstrad CPC. Es un sistema operativo de interfaz gráfica y multitarea

iPhone OS

2007

Fue un sistema operativo móvil, que llego con su primer celular revoluciono el mundo con su interfaz gráfica táctil.

Android 1.0

2008

Se desarrolla el primer modelo comercial, presentaba características como una interfaz grafica agradable con el usuario, y permitía descargar apps mediante la app store como YouTube, Gmail, etc

iPhone OS 2.0

2008

Llega a los celulares la app store, ofreciendo nuevas apps.

Windows 7

2009

Un sistema operativo con un mejor rendimiento, estabilidad e interfaz gráfica.

Android 1.5 Cupcake

2009

Entre las cosas más importantes podemos encontrar: se agregan nuevos widgets, se introduce el teclado, pero ahora en la pantalla y un nuevo soporte de emparejamiento bluetooth

IOS 4

2005

cambia su nombre de IPhone OS a IOS, trajo nuevas características en sus sistemas como añadir carpetas en su pantalla principal, nuevos fondos de pantalla, agrego la función multitarea, ubicación, entre otros.

Android 3.0 Honeycomb

2011

El primer Android para tablets, se mejoran la interfaz graficas, soporte para widgets, etc.

IOS 5

2011

Agregan a Messenger, una inteligencia llamada Siri y un centro de notificaciones y también agrego ICloud un servicio que guarda los datos en la nube.

Windows 8

2012

Llega a los celulares la app store, ofreciendo nuevas apps.

OS X Mountain Lion

2012

Es el sistema operativo de Apple, integran iCloud, Imenssage y notas, entre otras. También agregaron un centro de notificaciones y recordatorios.

IOS 7

2013

Se rediseñan la interfaz grafica y estilo de las apps, además es el sistema operativo mas seguro del momento

Android 5.0 Lollipop

2014

Mejora considerablemente su apariencia, mejora las notificaciones, rediseña el estilo de sus nuevas aplicaciones y ofrece un sistema multiusuario.

Windows 10

2015

Combino el sistema en PC, Tablets y Teléfonos. debido a las críticas por eliminar el menú de inicio se volvió a introducir, también mejores cuestiones de seguridad y ofrecía actualizaciones gratuitas

MacOS Sierra

2016

Es el sistema operativo de Apple, cambia de nombre de OS X a Mac OS, sumándose a IOS. Utiliza iCloud para sincronizar los datos, y añade a Siri a su barra principal

Android 10

2019

Se dejan de utilizar nombres de postres, empieza ofrecer mayor privacidad, personalización del teléfono, completar tareas y se implementa el sistema de navegación por gestos.

IOS 14

2020

Rediseñan los widgets y permiten que se añaden a la pantalla de inicio, se añaden nuevas apps a la biblioteca y la privacidad sigue mejorando.

Mac OS Big sur

2020

Rediseñan completamente la interfaz grafica, le agregan potencia a su escritorio y belleza siendo uno de los mejores sistemas operativos. También empiezan la transición de procesadores del Intel a los Apple silicon (M1)

Android 12

2021

Es el sistema mas personal hasta ahora, personalización dinámica de colores, renovación de widgets. Hasta este momento el Android más recomedle para teléfonos y celulares. /p>

Windows 11

2021

Mejoro su interfaz gráfica, mejoro notablemente el rendimiento de las computadoras, rediseño sus widgets y también su barra de navegación la hizo más minimalista agregando información sobre el tiempo, etc.

IOS 17

2023

Se mejora la transferencia de datos con aidrops, se agregan nuevos emojis, y características de transcripción y por último corrigen errores de actualizaciones de seguridad.

MacOS Sonoma

2023

Agrega nuevos widgets, su rendimiento es mucho mejor, tiene una mayor personalización en cuanto a su escritorio, se mejor las videollamadas, y una mayor compatibilidad con sus procesadores M1, M2, M3.
